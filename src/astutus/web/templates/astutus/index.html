Astutus From the Browser¶

Astutus uses a custom extension to Sphinx to provide dynamic HTML pages. This approach makes it easy to add instructions and guidance functional pages. It makes it easy to get a good-enough look-and-feel, without a lot of investment in custom styling. It provides a high degree of integration between end user functionality, system documentation, and development planning and execution.

Current Capabilities:

  • Raspberry Pi’s
    • Adopted Pi’s
    • Search for Pi’s on LAN
  • USB Devices
    • View USB devices as a tree
    • Manage device aliases
    • Manage device names
  • Control Logging
